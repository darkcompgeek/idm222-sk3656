Persona Q2 User Interface Recreation

The UI Design of the Persona series has always inspired me. I wanted to recreate the interface of Persona Q2, but with me and my friends as characters in it. This project reimagines Graphic Design critiques as a video game.

I created all the assets in Adobe Illustrator and Sketch. I animated this project entirely in HTML, CSS, and Javascript (using Greensock).
