Tesla Dashboard User Interface

For this project, we had to design a dashboard user interface for a car of our choice and animate it appearing after the car is turned on. I chose a Tesla. I designed the UI around two panels: the Driver Panel and the Passenger Panel. The driver panel contains all the essential information like speed and battery as well as essential car controls like lights and wipers. The Passenger Panel contains the more interactive “smart” elements of the car, such as navigation, music, and phone calls.

The project was animated using HTML, CSS, and Greensock.
